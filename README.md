# BioMonitoringOnTheAndes

A research-oriented repository focused on environmental and spectral analysis workflows for Andean ecosystem monitoring using remote sensing and computational methods.

---

## Overview

This repository contains notebooks, preprocessing workflows, and experimental evaluations related to bio-monitoring and spectral analysis in Andean environments. The project combines computational analysis, environmental indicators, and remote sensing techniques for exploratory research and future model development.

Current contents include:

- Data exploration notebooks
- Experimental evaluation workflows
- Spectral and environmental variable analysis
- Preliminary processing pipelines
